Bloqueador de segurança para liberação e proteção de aparelho

JP Secure Lock protege o dispositivo com PIN, criptografia AES-256-GCM e resposta anti-roubo: após 3 senhas erradas, executa wipe criptográfico e factory reset oficial do Android.

Criptografia E2E no aparelho

AES-256-GCM + PBKDF2-HMAC-SHA256. Chaves derivadas do PIN; cofre local sem servidor.

Bloqueador anti-roubo

3 PIN errados → limpeza de chaves, lockNow e factory reset (Device Admin oficial).

Desativar após liberar

Com PIN correto, libere o aparelho e desative a proteção + remova o administrador.

Recuperação

Recovery code gerado na ativação. Use se o wipe do cofre ocorreu sem factory reset.

Como funciona

  1. Instale o APK e abra o JP Secure Lock.
  2. Ative o Administrador do dispositivo (tela oficial do Android).
  3. Defina o PIN e marque o modo anti-roubo máximo.
  4. Guarde o recovery code offline (exibido só uma vez).
  5. Em roubo/tentativa: 3 erros = apaga todos os dados do aparelho.
  6. Após uso legítimo: desbloqueie e libere/desative.

O que é (e o que não é)

Item Realidade
3 erros → factory reset Sim (Device Admin + consentimento)
Eficaz contra roubo de dados Sim — aparelho volta “como novo”
Queima RAM / destrói hardware Não — impossível e não legítimo
Uso em aparelho de terceiros Proibido sem consentimento (crime)

Aviso: no modo máximo, 3 PIN errados apagam fotos, apps e contas. Faça backup. Teste com cuidado.
