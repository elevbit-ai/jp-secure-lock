Android · Anti-Theft · E2E Crypto

Security blocker for device protection and controlled release

JP Secure Lock uses a user PIN, AES-256-GCM encryption, and Android Device Admin. After 3 wrong passwords, it wipes crypto keys and can factory-reset the phone — effective against data theft.

Download signed APK v1.1.0 GitHub repository

Created by Joaquim Pedro de Morais Filho · br.com.joaquimpedro.jpsecure

How it works

  1. Install the APK and open the app.
  2. Enable Device Administrator.
  3. Set PIN and maximum anti-theft mode.
  4. Save the one-time recovery code.
  5. 3 failures → cryptographic wipe + factory reset.
  6. Legitimate unlock → release and deactivate.
